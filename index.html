Executive Summary

Project Overview

UX design and research for PDE Portal - a comprehensive platform for managing customer data, user registrations, and order processing in the healthcare industry.

Target Users

System Administrators, Organization Admins, and End Users requiring seamless access to healthcare portal functionalities.

Project Timeline

Completed: June 15th, 2025
Client: McKesson
Industry: Healthcare/Pharmaceutical

UX Research Methodology

Stakeholder Interviews

In-depth discussions with system admins, organization admins, and end users

User Interviews

One-on-one sessions to identify pain points and challenges

Surveys

Quantitative data collection on user expectations and frustrations

Competitive Analysis

Analysis of similar healthcare systems and industry best practices

Key Findings

  • Account Management Confusion

    Users struggled with complex mapping between different accounts (Sold-to, Ship-to, Bill-to)

  • Order Processing Frustrations

    Slow and error-prone order placement process with lack of feedback during validation

  • Inefficient User Onboarding

    Difficulty with self-sign-up process and OKTA credential integration

Solution Strategy

Design Approach

User-centered design with wireframing, prototyping, and multiple rounds of usability testing to ensure optimal user experience.

Key Features

Simplified registration flow, Order Wizard, integrated notifications, and seamless OKTA login integration.

Design Principles

Consistency, accessibility, real-time feedback, and error prevention throughout the user journey.

Quantifiable Results
